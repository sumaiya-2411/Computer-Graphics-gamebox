## 🚀 Featured Game: Infinite Neon Track Game

A futuristic infinite runner game built with Python and OpenGL.

### ✨ Features
- Endless gameplay  
- Neon-style graphics  
- Increasing difficulty  
- Collectibles and obstacles  

### 🛠️ Tech Used
- Python  
- OpenGL  
- GLUT  

### ▶️ How to Run

```bash
pip install PyOpenGL PyOpenGL_accelerate
